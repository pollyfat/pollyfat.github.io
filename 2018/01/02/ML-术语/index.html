A

accuracy 精度

对分类预测

accuracy = 分类准确个数 / 总测试数目

对判断型预测

accuracy = (判断为’是’并预测准确的个数 + 判断为’否’并预测准确的个数) / 总测试数目

activation function 激活函数

对输入[数组]进行加权求和, 并对其做一定处理后输出到下一层

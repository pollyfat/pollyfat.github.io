    ASCII Art:使用Graph::Easy绘制纯文本流程图

    Aniao
    2017-03-06

    install

    MAC

    -> brew install graphviz 
-> cpan
-> install Graph::Easy

    Windows

    • install graphviz(graphviz)
    • install perl(perl)
    • install cpan(cpan)
    • install Graph::Easy by cpan

    Use

    graph-easy someDSL.txt

    Grammer

    annotations

    # annotation
#illegal annotation
#FFFFFF # color
\#transferred meaning

    Node

    [single node]
[node a],[node b]

    Edges

    1. ->
2. =>
3. .>
4. ~>
5. .->
6. ..->
7. = >
8. - >

    add label:

    [ node a] - label ->[node b]

    result:

    +--------+  request   +--------+
| client | ---------> | server |
+--------+            +--------+

    Attributes

    {attribute: value}

    Samples

    Table:
    [VER|NMETHODS|METHODS||1|1|1 to 255]

    result:

    +-----+----------+----------+
| VER | NMETHODS |  METHODS |
+-----+----------+----------+
|  1  |     1    | 1 to 255 |
+-----+----------+----------+
